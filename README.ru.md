# planar-echo

\[Russian\] [\[English\]](README.md)

Инструмент, позволяющий запускать игры на движке Infinity Engine в браузере.

Требуется локально установленная оригинальная игра.


## Юридическая сторона

Так делать законно, если у вас куплена оригинальная игра.

Planar-echo не работает без оригинальной игры. Вся работа выполняется локально: данные никогда не покидают ваш компьютер.

Благодаря этому данный инструмент не связан с пиратством: planar-echo не содержит и не распространяет объекты авторского права.

Относитесь к planar-echo как к эмулятору.


## Статус

Техническая демонстрация.

Planar-echo находится в активной разработке.


### Что уже работает

- Поддерживаемые игры:
  - Planescape: Torment (только enchanted edition)
- Конвертация оригинальных файлов игры в json
 - .cre
 - .dlg
 - .eff
 - .ids
 - .ini
 - .itm
 - .tlk
- Полная поддержка русской и английской локализаций
- Отрисовка диалогов в браузере (без логики)


### Ближайшие планы

- Cz, de, fr, ko, pl переводы подготовлены - попросить сообщество проверить их на адекватность
- Реализовать логику в диалогах
- Добавить упаковку json обратно в biff, чтобы помочь сообществу моддеров
- Предоставить готовые артефакты .sh/.exe/.apk с контентом игры, чтобы не переконверчивать заново


### О лицензировании

Planar-echo состоит из пяти частей:

| Название         | Описание       | Лицензия                              |
|:-----------------|:---------------|:-------------------------------------|
| planar-asclepius | бэкенд          | по лицензии репозитория |
| planar-ghost     | артефакт сборки | по лицензии оригинальной игры <br /> не регулируется лицензией репозитория |
| planar-prism     | парсер          | по лицензии репозитория |
| planar-shared    | общий пакет     | по лицензии репозитория |
| planar-shell     | фронтенд        | по лицензии репозитория |


## Как работает (кратко)

1. Вы предоставляете оригинальную игру
2. В интерфейсе planar-shell настраиваете процесс конвертации через сервер planar-asclepius
3. Planar-asclepius запускает planar-prism, чтобы разобрать оригинальные файлы игры в json и js файлы
4. Planar-shell выполняет js исходники, которые предоставляет planar-asclepius


## Как запустить


### С docker

1. Предоставьте оригинальную игру
1. `docker compose build`
1. `docker compose up`

Запустятся одновременно бэкенд и фронтэнд через реверс прокси.

### Без Docker

1. Предоставьте оригинальную игру
1. Установите [nodejs](https://nodejs.org/)
1. Установите [yarn](https://yarnpkg.com/)
1. Запустите `./start.sh` на linux или `./start.ps1` на windows
1. Откройте `http://localhost:3003/`


## Как внести вклад

Задачи и пул-реквесты на [GitHub](https://github.com/snowinmars/planar-echo/).

```bash
# установите все пакеты, кроме planar-shared
yarn --cwd ./planar-asclepius
yarn --cwd ./planar-prism
yarn --cwd ./planar-shared
yarn --cwd ./planar-shell

# установите пакет planar-shared
yarn --cwd ./planar-shared start
yarn --cwd ./planar-asclepius add file:../planar-shared --force
yarn --cwd ./planar-prism     add file:../planar-shared --force
yarn --cwd ./planar-shell     add file:../planar-shared --force

# в одном терминале запустите фронтенд
# он будет работать на захардкоженном URL http://localhost:3000
# и использовать захардкоженный URL бэкенда http://localhost:3003
yarn --cwd ./planar-shell start

# в другом терминале запустите бэкенд
# он будет работать на захардкоженном URL http://localhost:3003
yarn --cwd ./planar-asclepius start
```
